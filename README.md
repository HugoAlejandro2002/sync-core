# Caja Inteligente Bolivia — Backend

Backend para estructurar evidencia financiera de comerciantes informales.

## Requisitos

- Python 3.12+
- [uv](https://docs.astral.sh/uv/)
- MySQL 8

## Instalación

```bash
uv sync
```

## Ejecución

```bash
uv run fastapi dev src/app/main.py
```

## Tests

```bash
uv run pytest
```
